029._ Curso de JavaScript para principiantes. Encapsulacion en JavaScript.

ENCAPSULAR ES ENCERRAR datos dentro de una clase.

En una funcion constructor Variables y funciones se convierten En Propiedades y Metodos usando la palabra clave "this"

Mediante el uso de la palabra clave "this" las variables se convierten en Propiedades y las funciones se convierten en Metodos y en ese caso si son accesibles desde las instancias objeto que creemos.

Funcion Privada: solo accesible Desde dentro de la Funcion Constructor a la que pertenece. -
