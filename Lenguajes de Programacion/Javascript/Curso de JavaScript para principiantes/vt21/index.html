021._ Curso de JavaScript para principiantes. El uso del Objeto Date. FECHA Y HORA Usos del Objeto Date. se trata de un Objeto que no forma parte del DOM, en este caso vamos a tener que crear nuestros Objetos Date. Y cada uno de los Objetos Date que creemos puede representar una FECHA y HORA diferente.
