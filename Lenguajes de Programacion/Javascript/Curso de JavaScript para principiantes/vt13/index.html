13.- Curso de JavaScript para principiantes. Pasar argumentos a funciones. Se pueden llamar a funciones pasandole datos, y estos datos que le pasamos a la funcion se llaman "argumentos" Parametro y argumentos NO son lo mismo. Parametro: Cuando declaramos la funcion tenemos que indicarle que tipo de datos y el numero que tenemos que pasarle y eso es lo que se denomina "Parametros". argumentos: Cuando llamemos a la funcion y le demos unos valores para cada uno de esos datos obligatorio es cuando se llama "argumentos". Asi que "parametro" es en la declaracion, "argumento" es en la llamada. pasar argumentos: Datos distintos que se pasan al mismo tiempo que se llama a la funcion.
