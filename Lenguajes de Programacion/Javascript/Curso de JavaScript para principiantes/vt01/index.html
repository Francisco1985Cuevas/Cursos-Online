01.- Curso de JavaScript para principiantes. ¿Para qué sirve JavaScript? vamos a iniciar este curso desde lo mas basico y cuando lo terminemos tendras una idea muy clara y tambien un dominio practico del trabajo tanto del lenguaje de programacion JavaScript como de la multitud de herramientas que permiten el desarrollo imparable que esta conociendo la web en la actualidad. - la primera pagina web solo contenia codigo HTML. - Lo primero a tener en cuenta es que HTML no es un lenguaje de programacion, sino que como nos indica su nombre HTML se trata de un lenguaje de "marcado" con esto nos referimos a que es un lenguaje que usamos para "marcar" partes de una pagina esto es lo que le indica al navegador que esas partes tienen que mostrarse de un modo particular. Por ejemplo si queremos que una pagina de nuestro texto aparezca en negrita en lenguaje de "marcado" entonces tenemos que establecer las "marcas" en este caso poner el "Tag"

al comienzo y al final, es a eso lo que nos referimos cuando hablamos de marcas. - Estas paginas por su propia naturaleza que es ESTATICA, no pueden responder a las acciones del usuario, tampoco pueden tomar decisiones y no pueden tampoco modificarse la visualizacion de los elementos que forman la pagina. Por tanto este marcado que contiene siempre sera interpretado y sera mostrado al usuario exactamente igual cuando sea visitada la pagina. - Hoy las paginas web no son nada ESTATICAS, estas capacidades que podemos llamar inteligentes en los navegadores se ponen al alcance del usuario gracias a programas que se suelen llamar "Scripts" y que operan en la sala de maquinas manipulando lo que se muestra en nuestro navegador. ¿Como podemos añadir esos Scripts? Existen dos modos fundamentales: - Podemos hacer que el Servidor Web ejecute un script antes de mostrar la pagina al usuario y en esos scripts se define la informacion que es enviada al navegador. El modo de funcionamiento de estos scripts es que se ejecutan antes de generar la pagina que va a ser enviada al navegador y mostrada al usuario. Y lenguajes como PHP, ASP, etc., son buenos ejemplos de ese tipo de lenguajes del lado del Servidor. los scripts son ejecutados por el navegador.
