El mapa de un documento web. - De momento sabemos que el codigo JavaScript puede mezclarse con el codigo HTML para modificar el comportamiento de una pagina web de muchos modos diferentes. La clave ahora esta en entender el modo en el que el codigo JavaScript y el codigo HTML interactuan entre si, y para eso es impresindible entender el Document Object Model. DOM: Document Object Model. DOM, Similar a un Plano Topografico. El DOM es la clave en la estandarizacion de los documentos web. - Cada Objeto del DOM es un elemento UNICO. - Cada vez que se le solicita al navegador que se le cargue una pagina web, lo primero que hace es interpretar, analiza todo el codigo fuente que contiene la pagina HTML y como parte de este proceso de interpretacion el navegador crea un tipo de modelo interno que se basa en el estandar DOM, es el que refleja el contenido completo del Documento.
